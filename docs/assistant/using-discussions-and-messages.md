# Using Discussions And Messages

## How to use Discussions
Use Discussions when you want structured public conversation around a topic, project, question, or enthusiast interest. Discussions are best when the conversation should stay visible and useful to more than one person.

## What people usually do in Discussions
- ask questions
- compare opinions
- discuss cars, projects, or categories
- share context that benefits the broader Carmunity

## What topics fit Discussions best
Discussions work best for:
- questions with public value
- enthusiast discussion and opinions
- project updates or topic-based threads
- broader auction or ownership conversation that should stay visible

## How Messages are different
Messages are the direct one-to-one layer. Use Messages when a public conversation needs to become private, direct, or more personal in scope.

## When to move from Discussions to Messages
Move from Discussions to Messages when:
- you need direct follow-up with one person
- the conversation stops being broadly useful as a public thread
- you need one-to-one coordination or clarification

## Forums and Discussions
If someone asks about forums, the Carasta public term is Discussions. The assistant should treat forum-style questions as questions about Discussions unless the user clearly means something else.
