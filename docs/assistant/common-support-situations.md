# Common Support Situations

## If the assistant cannot verify your situation
Use Contact when the issue depends on private account state, support review, balances, approvals, disputes, or anything else the assistant cannot safely verify.

## If the question is about trust or safety
Use Trust and Safety first when the question is about:
- platform boundaries
- safety
- moderation-related guidance
- what Carasta is or is not responsible for

## If the question is about selling workflows
Use the seller and selling guides when the question is about:
- seller tools
- listing workflows
- seller AI
- the seller growth workspace

## If the question is about buying or auctions
Use the auction and buying guides when the question is about:
- bidding
- reserve state
- anti-sniping
- buyer expectations

## If the question is about community participation
Use the Carmunity, Discussions, and Messages guides when the question is about:
- public conversation
- private messaging
- forum-style questions
- how community surfaces relate to each other
